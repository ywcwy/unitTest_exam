# unitTest_exam

## Environment
* Node.js

## Packages
* mocha
* chai

## Install (for Mac OS user)
### 1. Open Terminal

### 2. Project Download
```
$git clone https://github.com/ywcwy/unitTest_exam.git
```
After download, 
```
$cd unitTest_exam 
```
### 3. npm Installation
```
$npm init -y    
```
### 4. Package Installation
```
$npm install     
```
### 6. Project Implement
```
$npm test
```
